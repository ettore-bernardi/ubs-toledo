# UBS Toledo

🏥 **Descubra qual Unidade Básica de Saúde (UBS) atende o seu bairro em Toledo/PR**

Uma aplicação web instalável, rápida e sem dependência de internet para ajudar cidadãos de Toledo a encontrar sua unidade de saúde mais próxima.

**[🚀 Acessar aplicação →](https://saude.ettore-bernardi.com/)** | **[📂 Código-fonte](https://github.com/ettore-bernardi/ubs-toledo)**

---

## 🎯 Para quê?

Você sabe qual é a UBS responsável pelo seu bairro em Toledo?

Esta aplicação responde:
- ✅ Qual UBS atende meu bairro?
- ✅ Qual é o endereço e telefone?
- ✅ Qual é o horário de funcionamento?
- ✅ Como chego lá do meu local atual?

---

## 🚀 Recursos Principais

### 📍 Mapa Interativo
- Visualize todos os bairros de Toledo no mapa
- Toque em um bairro para ver qual UBS o atende
- Mapa funciona offline após a primeira visita

### 🔍 Busca Rápida
- Procure seu bairro pelo nome
- Lista completa e pesquisável
- Resultados em tempo real

### 📞 Contatos Diretos
- **Telefone:** Clique para ligar direto
- **WhatsApp:** Abra conversa no WhatsApp
- **Email:** Envie mensagem por email
- **Horários:** Confira os dias e horas de atendimento

### 📱 Instalável no Celular
- Instale a aplicação na tela inicial do seu celular (Android e iOS)
- Acessa como um aplicativo nativo
- Funciona offline após a instalação

### 🌐 Sem Dependência de Internet
- Depois da primeira visita, tudo funciona offline
- Economiza dados móveis
- Rápido e responsivo mesmo com conexão lenta

## 💻 Como Usar

### Opção 1: Abrir no Navegador
1. Acesse [saude.ettore-bernardi.com](https://saude.ettore-bernardi.com/) no seu navegador
2. Toque em um bairro no mapa **ou** procure na lista
3. Veja os detalhes da UBS: endereço, telefone, horário
4. Clique em "Como chegar" para abrir o mapa de rotas

### Opção 2: Instalar no Celular

#### Android
1. Abra a aplicação no Chrome
2. Toque no ícone ⋮ (menu) no canto superior direito
3. Selecione "Instalar app"
4. O aplicativo será adicionado à sua tela inicial

#### iPhone/iPad
1. Abra a aplicação no Safari
2. Toque no ícone de compartilhar (↗️)
3. Role para baixo e selecione "Adicionar à Tela Inicial"
4. O aplicativo funcionará como um atalho rápido

## ⚠️ Importante

A atribuição de bairro para UBS é feita por **proximidade** (distância até a unidade mais próxima). 

**Sempre confirme por telefone ou WhatsApp antes de se deslocar até a unidade**, pois:
- Sua UBS de referência pode ter mudado
- Alguns bairros podem ser atendidos por mais de uma unidade
- Dados podem estar desatualizados

Temos o telefone/WhatsApp sempre em destaque no card da UBS justamente para facilitar essa confirmação!

## 🔧 Características Técnicas

- **Sem servidor backend:** 100% funcionamento offline-first
- **Dados privados:** Nenhuma informação sua é coletada ou enviada para servidor
- **Rápido:** Carrega em segundos, mesmo em conexão lenta
- **Leve:** Apenas ~70 KB de dados após compressão
- **Moderno:** PWA (Progressive Web App) com suporte em Android 5.0+ e iOS 13+
- **Acessível:** Otimizado para leitores de tela e navegação por teclado

## 📝 Dados

### Fonte dos Dados
- **Bairros:** Mapa oficial da prefeitura de Toledo (GeoServer)
- **UBS:** Registros da Secretaria Municipal de Saúde
- **Contatos:** Informações públicas da prefeitura

### Atualizações
Os dados são atualizados periodicamente. Se notar informação desatualizada:
1. Clique no WhatsApp da UBS para confirmar
2. Abra uma issue neste repositório com a correção
